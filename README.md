# TorSurf
- TorSurf is a similar project as ParrotSec's Anonsurf.

- Since Anonsurf's dependencies either not available or a challenge to install on Fedora, TorSurf serves as an alternative to AnonSurf for Fedora/dnf based distros.

- For Debian/apt based distros, refer to this project : [kali-anonsurf](https://github.com/Und3rf10w/kali-anonsurf)
